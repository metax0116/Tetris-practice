Tetris Architecture Documentation

This Tetris application is built with React and TypeScript, focusing on a hook-based state management pattern.

Core Components

  • App.tsx: Main component. Handles keyboard events and renders the UI.
  • useTetrisGame.ts: The "Brain" of the game. Manages all state (stage, player, score) and logic (movement, rotation, collision).
  • useInterval.ts: Custom hook for the game loop (gravity/drop timing).
  • tetriminoes.ts: Utility defining shapes, colors, and piece generation.

Data Flow

  1. Input: User presses a key. App.tsx catches it and calls a method from useTetrisGame.
  2. Update: useTetrisGame updates the player position or stage state after checking for collisions.
  3. Tick: useInterval triggers drop() every X milliseconds, moving the piece down automatically.
  4. Render: App.tsx re-renders the stage, merging the static grid with the active player piece.
